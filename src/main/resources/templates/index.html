Kenapa NetraSphere?

Teknologi Level Enterprise

Bukan sekadar CRUD — platform ini dirancang untuk skala komunitas nyata dengan arsitektur microservice-ready.

🔐

RBAC Tiga Peran

Admin, Citizen, Collector — setiap role punya akses berbeda. Diperkuat JWT + Spring Security.

💚

Green Wallet

Setiap setoran menghasilkan poin. Tiga multiplier berdasarkan jenis sampah: 1x, 1.5x, 2x.

🤖

IoT NetraDUMP

Endpoint /api/v1/iot/dump siap menerima data dari robot pengumpul otonom.

🧠

AI Chatbot Mistral

Tanya panduan daur ulang atau estimasi poin langsung ke AI. "Berapa poin untuk 2 kg plastik?"

Alur Sistem

3 Langkah Sederhana

🗑️
STEP 1

Pilah & Setor Sampah

Warga memilah sampah berdasarkan kategori (Organik, Anorganik, B3) lalu menyetorkan melalui app atau smart bin.

✅
STEP 2

Konfirmasi Petugas / Robot

Petugas atau robot IoT NetraDUMP mengkonfirmasi berat sampah. Sistem menghitung poin secara otomatis.

🎁
STEP 3

Tukar Poin dengan Reward

Poin Green Wallet bisa ditukar dengan voucher belanja, diskon tagihan, atau donasi lingkungan.

IoT Integration

Siap Terhubung ke Robot NetraDUMP

Endpoint khusus IoT memungkinkan smart bin atau robot pengumpul otomatis mengirim data tanpa UI manual. Cukup gunakan API key dan kirim POST request.

✓Auto-konfirmasi tanpa intervensi manusia
✓Poin otomatis dikreditkan ke Green Wallet
✓Audit trail lengkap per device ID
✓Data lokasi GPS / nama titik pengumpulan
POST /api/v1/iot/dump 
curl -X POST \
  https://api.trinetra.com/api/v1/iot/dump \
  -H "X-IoT-Api-Key: IOT-TRINETRA-NETRADUMP" \
  -H "Content-Type: application/json" \
  -d '{
  "deviceId": "NETRADUMP-001",
  "citizenId": "warga-uuid-here",
  "categoryId": "plastik-pet-uuid",
  "weightKg": 2.5,
  "location": "Smart Bin Area A"
}'

// Response:
{
  "success": true,
  "message": "Setoran dikonfirmasi. Poin: 37",
  "data": {
    "status": "CONFIRMED",
    "pointsEarned": 37.5,
    "fromIoT": true
  }
}
